What is CyberPop?

Welcome to CyberPop!
This website has been created in 2020 in order to help people from all around the world with understanding the main concepts of technology, using "lessons" as a way to teach them.

"Lessons" can be found on CyberPop's main page.
Did you know that what you are reading right now is also considered as one?

They are mainly composed by paragraphs and images.
The purpose of images may differ depending of the type of lesson, but in the case of Web Development's HTML lessons, images are screenshots of HTML and lead to the result of that same HTML when clicked onto!

For example, below this can be found the image used in the third HTML lesson of the Web Development category.

As CyberPop aims to help people from all around the globe, it is built to be available in multiple languages.
The entirety of the website, from menus to lessons, is currently available in both English and French!

The website is in constant development in order for it to feature more lessons and more features.
Check back every now and then!
