Headings, sections and attributes

Sometimes, when you make a HTML file, there is text to which you want to give more importance than others. That is what headings are for!
But what's more important than important text are <section> elements, and the fact that pretty much all of the elements in HTML can have attributes, which can be used to customize them!
To begin with, let's see a HTML document which features all three of those things.

But! Before we talk about what was previously mentionned, we'll check out two really easy elements that are featured in this HTML file, namely <header> and <footer>.
<header> features all the things that should show up at the top of the page, while <footer> features all the things that should show up at its bottom. If you have visited many websites, you may have already noticed that the header and the footer elements are usually the same throughout all webpages of a same website!

Alright, now, headings. There are 6 headings element in HTML, from <h1> to <h6>, each of them make its content bold and changes its size (differently) without the need for CSS.
<h1> makes the text quite big, <h4> doesn't really change the text size, while <h6> makes it really small. Remember that all of them makes the text bold.

Now, <section>. This element is basically a box of whatever size which you can put wherever in which you can put whatever. You can even put a <section> inside another <section>, or put it in a page's header or footer! Or just inside the <body>, that's fine too.

Something I must mention is that, while there should only be one <!DOCTYPE html>, one <html>, one <head> and one <body>, there can be as many <p>, <h1>, <h6>, <section> and many other elements as you want!

And to about any element you want, you can give as many attributes as you want! In the example, only the style attribute is given, thrice, one to the only <section> element, one to the <p> element that happens to be inside that same <section> element, and the <h4> element.
The style attribute is used to apply CSS directly into the HTML, instead of using CSS files, with a syntax of style="property: value" inside the start tag.

The style of <section> is there to make it so every text inside this same <section> becomes red.
The one of <p> makes it so its content gets aligned to the right. (put simply)
<h4>'s made so it gets centered to the middle.

Simple, right? You should now understand why the "Hello World" paragraph is both red and aligned to the right, and how each element is affected by its own position!
