The initial structure of your own HTML document

The webpage you are visiting right now is most and foremost a file with the .html extension with valid content so it can be rendered by whatever web browser you are currently in, no matter its window size.
This file is hosted somewhere in the world on a computer, and you can see of what it is made by right-clicking on it in your web browser, and choosing the "Inspect element" option.
However, you will hardly understand anything as you don't really know anything about HTML yet, so let's first see what every HTML files' content have in common!

The very first element is <!DOCTYPE html> which is something needed at the top of every HTML file to make it somewhat "valid". Don't question how come it is necessary, just remember it is, ok?
Before the fifth and current version of HTML, it used to be more complex: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

The second element is <html>, but may I take your attention to the very last line of the file, as it is related to this element? It is written </html> there. Basically, everything after <html> and everything before </html> is where everything will be written in, with the only exception of <!DOCTYPE html> which you have already met. Anything else that is not written between those two is simply invalid.

By the way, everything that is written like <something> is called a start tag, while everything that is written like </something> is called an end tag. Together, they form an element! Some rare elements only need a start tag.

The third element is the <head> element, it's basically where you're gonna put about everything related to the webpage that isn't meant to be visible ever in it, such as the title, the fonts, the character encoding and sometimes, the scripts necessary for interaction.

In this example, the <head> element contains the fourth element we're gonna talk about, and it is the <title> one! Its purpose is straight-forward, and how to use it is really simple. Pretty much every webpage you visit in your web browser has a <title> element, and it is the text between the start tag and the end tag that makes its tab name.
So, this here webpage has a <title> element of "The initial structure of your own HTML document" and if you click on the image above that shows the exemple, you will see that the webpage's tab name would be "CyberPop!", what a cool name.

The next element is the <body> one, which is yet again a very simple one, as it can be considered as the direct opposite to <head>. It is where you put all the elements that are or will become visible.
And you also can put your scripts there, it's up to preference.

The last element featured in this example conveniently happens to be in the <body> element. Let's talk about the <p> element!
This is with this element that you put paragraphs, or text with less importance than your webpage's titles, usually called "headings". More on those for another time!
