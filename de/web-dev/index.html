Jede Webseite die du besuchst ist mehr oder weniger ein HTML Dokument.

CSS

CSS sorgt für die Gestaltung und richtige Anordnung!
Es gibt dem Browser Informationen wie er das HTML Dokument anzeigen soll. Dabei gibt es Anweisungen über Form, Farbe und Anordnung der Inhalte.
