DÉVELOPPEMENT WEB

HTML

Chaque page web est en tout premier lieu un fichier HTML.
C'est ce fichier qui décrit au navigateur web la structure de la page web, il peut être comparé à un squelette !

CSS

Le CSS est ce qui rend une page web belle et organisée !
C'est ça qui change comment le navigateur web affiche les différents elements de la page web, leurs formes, positions et couleurs.

JS

Les pages web sont généralement meilleurs lorsqu'elles sont interactives.
C'est l'une des nombreuses raisons pour laquelle le JS est utilisé en développement web !
