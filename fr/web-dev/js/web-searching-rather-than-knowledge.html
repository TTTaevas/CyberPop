Dépendre de la recherche Internet plutôt que de vos connaissances

Vous avez probablement remarqué qu'il y a beaucoup d'éléments en HTML, beaucoup d'attributs et de valeurs en CSS ainsi que beaucoup de fonctions en JS. Vous pensez peut-être que vous devez absolument tout savoir de ces trois langages afin de créer une page web qui ressemble à quelque chose, mais ce n'est heureusement pas le cas.

Ce que vous devez savoir est la syntaxe de ces trois langages, leurs concepts ainsi que leur rôle dans une page web, et c'est exactement ce que CyberPop vous a appris !
Mais même si vous connaissez la syntaxe et les concepts de ces langages, ça ne veut pas nécessairement dire que vous serez capable de les appliquer.

Vous allez avoir des idées qui utilisent les concepts que vous avez appris, mais comment appliquer des idées est relativement difficile et diffère en fonction du langage.
(C'est quand même encore ok en développement web si le seul langage de programmation que vous utilisez est Javascript !)

Par exemple, disons qu'en HTML, vous avez crée quelques éléments <p>, tous dans des classes différentes. Comment créer une fonction en JS qui obtient tous les éléments <p> qui ont une certiane classe afin de les rendre légèrement plus grand est quelque chose que vous ne savez probablement pas faire en pratique, bien qu'il est possible que vous ayez une théorie sur comment faire ça.

Genre, il y a probablement une fonction en JS qui obtient tous les éléments qui ont cette description, et vous pouvez faire une "boucle for" (appelé "for loop" en Anglais) afin de traverser une array de ces éléments, mais est-ce que vous vous souvenez de comment on fait une "for loop", et est-ce que vous savez quelle fonction en JS obtient tous les éléments d'une page web ayant une certaine classe ?

Il n'y a véritablement aucune honte à ne pas savoir, honnêtement. Comme j'ai dit plus tôt, vous ne devez pas absolument savoir toutes les fonctions JS !
Mais du coup.. comment appliquer votre idée si vous ne savez pas comment faire ?..

L'une des compétences que vous allez acquérir en faisant une page web (ou un logiciel) est la recherche web, en utilisant des moteurs de recherche comme Google, DuckDuckGo, vraiment, n'importe quel moteur qui vous donne les résultats que vous voulez, et vous pouvez en utilsier autant que vous voulez, c'est vraiment votre choix personnel !

Quelles phrases utiliser lors des recherches web est aussi votre choix personnel, mais je recommande mettre le nom du langage en premier, puis le concept et enfin, ce que vous voulez faire avec le moins de mots possible. Un exemple serait "JS array retirer dernier élément".

Je me dois de préciser que vous obtiendrez à peu près toujours les meilleurs résultats dans la langue Anglaise. Apprendre l'Anglais est quelque chose qui vous sera utile (notamment) en informatique, mais ça ne devrait pas être quelque chose de nécessaire pour vous pour l'instant.

Et enfin, quels résultats visiter. En Français, la documentation de Mozilla devrait être votre première priorité.
En Anglais, n'importe quel lien menant à Stack Overflow sera votre priorité, suivi par la documentation, qui, en développement web, sera toujours sur W3Schools et sur le site de Mozilla, sinon généralement sur les sites web gérés par les gens qui sont derrière ce avec quoi vous avez un soucis.
Par exemple, si vous utilisez C#, vous devriez jeter un oeil au site web de Microsoft ! (FR)

Cette leçon s'applique à n'importe quel langage, même pour le développement de logiciels. Constament faire des recherches sur le web est nécessaire pour faire quelque chose dont vous serez fier !
