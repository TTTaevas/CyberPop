DÉVELOPPEMENT WEB

HTML

Chaque page web est en tout premier lieu un fichier HTML.
C'est ce fichier qui décrit au navigateur web la structure de la page web, il peut être comparé à un squelette !

CSS

Le CSS est ce qui rend une page web belle et organisée !
C'est ça qui change comment le navigateur web affiche les différents elements de la page web, leurs formes, positions et couleurs.

JS

Les page web sont généralement meilleurs lorsqu'elles sont interactives.
C'est l'une des nombreuses raisons pour laquelle le JS est utilisé en développement web !

Hypertext Markup Language

  • La structure initiale de votre document HTML
  • Titres, sections et attributs
  • Images, liens et commentaires

Cascading Style Sheets

  • Faire en sorte que le HTML utilise votre document CSS
  • Prendre le contrôle de la taille des éléments HTML grâce au CSS
  • Bien comprendre le concept de priorité en CSS

Javascript

  • Exécuter du JS en cliquant sur un élément HTML
  • Bien comprendre le concept des variables, conditions et fonctions
  • Dépendre de la recherche Internet plutôt que vos connaissances
