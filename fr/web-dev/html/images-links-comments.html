Images, liens et commentaires

Les visuels sont importants. Et pas seulement du côté de ce que l'utilisateur voit lorsque son navigateur web charge la page, mais aussi du côté de l'intérieur du fichier HTML !
Jettons un oeil à comment ajouter des images et des liens du côté du navigateur web, et comment ajouter des commentaires pour mieux organiser nos fichiers HTML.

En temps normal, vous voulez ajouter une image à votre site web en utilisant la balise <img>. Mais l'utiliser seule est inutile si vous n'y ajoutez pas les attributs nécessaires ! L'élément <img> n'a pas de balise de fin, ce qui veut dire que vous avez besoin d'utiliser des attributs afin que le navigateur web montre l'élément <img> comme une image.

Tout d'abord, vous voulez dire au fichier HTML quelle est l'image que vous voulez que le navigateur web montre. C'est pour cette raison que l'attribut src existe ! La valeur que vous lui donneriez serait soit une URL menant à un fichier d'image, soit la position du fichier relative à celle du fichier HTML. Je pense que la deuxième option sera un peu trop compliquée pour l'instant, donc concentrez-vous sur l'utilisation de l'attribut src avec des URLs jusqu'à ce que vous décidez d'apprendre proprement l'HTML.
La syntaxe est très simple, elle est quelque chose comme "src='url'".
Pour l'instant, votre balise <img> ressemblerait à quelque chose comme <img src="https://www.w3schools.com/images/lamp.jpg">, mais il lui manque toujours un attribut.

Voici l'attribut alt ! Parce qu'un navigateur web pourrait ne pas bien "interpréter" l'image pour je ne sais quelle raison, par exemple des soucis de compatibilités ou de connexion, cet attribut existe.

Quand un navigateur web n'arrive pas à interpréter une image, il va tenter d'interpréter à sa place la valeur de alt, qui serait du texte.
Vous devriez toujours mettre du texte qui décrit qu'est ce que l'image qui aurait dû être affichée en tant que valeur de alt.

Votre navigateur web n'a pas réussi à vous montrer l'image ci-dessus car elle n'existe tout simplement pas. C'est dommage, vous n'avez pas pu voir à quel point le chat était mignon.

Dans l'exemple de l'image tout en haut de la page, il y a deux éléments sur lesquels vous pouvez cliquer. Si vous choisissez de cliquer dessus, votre navigateur web ouvrira un lien. Ce serait grâce à l'élément <a> !
Tout ce qui est à l'intérieur de cet élément envoiera l'utilisateur sur une autre page web (ou ailleurs sur la même page web) lorsqu'il lui cliquera dessus !

L'élément <a> requiert un attribut, plus spécifiquement href.
Comment cet attribut marche est très simple, car il marche comme src, l'attribut utilisé pour la balise de départ <img>. Sa valeur est l'URL que vous voulez visible d'une façon ou d'une autre par l'utilisateur. C'est comme ça que ça marche dans l'exemple aussi !

Mais dans l'exemple, quand vous cliquez sur l'image, le navigateur web ouvre l'URL dans un nouvel onglet, alors que lorsque vous cliquez le texte blue, il ouvre l'URL dans le même onglet !
Il y a en fait un attribut optionnel pour l'élément <a>, que l'on appelle target.

target, pour la balise de départ <a>, n'a qu'une seule valeur que vous devriez retenir pour le moment : _blank. Ajouter cet attribut avec cette valeur fait que le navigateur web ouvre le lien dans un nouvel onglet, tout frais !
Vous pouvez mettre autant d'éléments que vous voulez dans l'élément <a>, et vous pouvez aussi y mettre du texte simple sans avoir besoin d'un élément <p>.

Sinon, avez-vous déjà entendu parler des commentaires ? Dans les fichiers HTML, ils sont utilisés pour dire des choses à n'importe qui qui lirait l'intérieut du fichier, et ne seront tout simplement pas interpreté par le navigateur web, ils seront ignorés à la place.
Dans l'exemple en haut de cette page, il y a trois commentaires non-interprétés. Ils peuvent être plus ou moins n'importe où, mais ils devraient être placés à un endroit où ils peuvent être dicernés des choses qui ne sont pas des commentaires.

Un commentaire commence avec <!-- puis finit avec -->. Il peut prendre autant de lignes que vous voulez !
Et c'est tout, du coup. Vous voyez que tout ça n'est pas si compliqué après tout !
